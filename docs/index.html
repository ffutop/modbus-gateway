Quick Start

1. Build

bash
git clone https://github.com/ffutop/modbus-gateway.git cd modbus-gateway go build -o modbus-gateway

2. Run

# Listen on port 5020, Device /dev/ttyUSB0
./modbus-gateway -p /dev/ttyUSB0 -s 9600 -P 5020 -v debug

Configuration

Use config.yaml for advanced settings.

# TCP Server  
tcp_address: "0.0.0.0"  
tcp_port: 502  
max_conns: 32  
  
# Serial/RTU  
device: "/dev/ttyUSB0"  
baud_rate: 19200  
data_bits: 8  
parity: "N"  
stop_bits: 1  
timeout: 500ms  
  
# RS485  
rs485:  
  enabled: true  
  delay_rts_before_send: 2ms
